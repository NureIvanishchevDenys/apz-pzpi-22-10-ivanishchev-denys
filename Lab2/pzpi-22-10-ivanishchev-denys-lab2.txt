Харків 2025
	Для створення мобільного застосунку використовувалась мова програмування «Kotlin» для розробки Android застосунка. Як середовище використовувався «Android Studio». 
	Також використовувався Android Jetpack для швидкого оновлення UI при зміні даних та для переходів між активностями.
	Для візуалізації місцезнаходжень станцій використано Google Maps API, що показує маркер місця, де встановлено станція.
	Для обміну інформації з сервером використовується REST API з використанням HTTP-запитів та Retrofit.
