Хід роботи
	Фронтенд був написаний, використовуючи бібліотеку React для побудови інтерфейсу. Також був використаний компонентний підхід для розділення відповідальності між сторінками. 
	Після входу користувача його роль зберігається у localStorage для перевірки його ролі, щоб відкрився доступ до адмін-панелі. Також захист адмін-панелі від звичайних користувачів зроблено через компонент, який перевіряє роль користувача.
	Сайт отримує інформацію через запити до REST API через fetch.
	Взаємодія системи один з одним:
•	IoT-пристрої передають інформацію про вимірювання у БД через API;
•	База даних зберігає інформацію про станції, користувачів, заміри, параметри в собі для подальшого експорту у мобільний додаток або сайт;
•	Мобільний додаток дає можливість продивитися інформацію про заміри станцій;
•	Веб-сайт також дає доступ до замірів, але також має адмін-панель для роботи з БД.
