	1. Мета роботи
	Внести зміни у Vision & Scope документ, який був створений, щодо розділу 3 Scope and Limitations.
	2. Хід роботи
	Назва проекту – Програмна система для моніторингу якості води «AquaSafe».
	Концепцією є створення пристрою, який у реальному часі відображає якість води у місці встановлення. Його встановлення забезпечить доступ до сучасного аналізу води навіть у віддалених місцях
	Метою є задоволення потреб екологічних організацій та приватних користувачів у компактному, точному та зручному рішенні. Пристрій сприятиме підвищенню екологічної обізнаності та безпеки завдяки своєчасній і точній інформації про якість води.
	Впровадження цього рішення дозволить зменшити залежність від громіздких і дорогих стаціонарних лабораторій та сприятиме швидшому реагуванню на екологічні проблеми.
3. Scope and Limitations
	3.1 Scope of Initial Release
	Реліз проекту включає систему моніторингу якості води, яка складається з таких компонентів:
•	IoT пристрій, що вимірює основні параметри води. Базується на мікроконтролерах Arduino;
•	Серверна частина на базі даних, написаній на MySQL та API для прийому та збереження даних від IoT-пристроїв;
•	Мобільний застосунок, який дозволяє переглядати поточні та історичні вимірювання якості води;
•	Веб-клієнт для розширеного перегляду даних, звітності та адміністрування IoT пристроїв.
	3.2 Scope of Subsequent Releases
	Робота зі штучним інтелектом дозволить виявити аномалії у даних в режимі реального часу для автоматичних сповіщень. Також ШІ може прогнозувати рівень забрудненості на основі історичних даних.
	Дані, які буде збирати ШІ, це самі дані та геолокацію сенсорів.
	Буде використовуватися готове API зі штучним інтелектом.
	3.3 Limitations and Exclusions
	У початкових версіях системи не буде реалізоване органічне забруднення бактеріями чи вірусами. Також система не буде призначена для роботи у високо- або низькотемпературних середовищах. 
	Навмисно виключеним функціоналом буде підтримка iOS через неможливість перевірки функціонування у реальному часі. Також не буде інтеграції з базами даних екологічних стандартів через складність реалізації.
